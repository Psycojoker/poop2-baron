Baron

  • ast lossless ! (FST == Full Syntaxe Tree)
  • source == ast_to_code(code_to_ast(source))
  • transforme un problème d'analyse de code en parcours/modification d'un graph
  • output du json pour compatibilité maximum (+ structure de donnée simple)

1 an de boulot (j'ai du apprendre)

  • +1000 tests (TDD)
  • marche sur le top 100 de pypi
  • utilities: position_to_path, position_to_node, boundinx_box, walker etc...
  • (encore quelques bugs ultra rare)

26/34

RedBaron: exemple

 1 from redbaron import RedBaron
 2 
 3 red = RedBaron("print 'Hello World!'")
 4 
 5 red[0]  # print 'Hello World!'
 6 
 7 red.find('print')  # print 'Hello World!'
 8 red.print_  # print 'Hello World!'
 9 red.string  # 'Hello World!'
10 
11 red.find_all('print')  # [print 'Hello World!']
12 red('print')  # [print 'Hello World!']
13 
14 red.string.value = "'Hello from RedBaron!'"
15 red  # print 'Hello from RedBaron!'
